# Expense Tracker

### Built with

<ul>
  <li>NodeJS</li>
  <li>Express</li>
  <li>PostgreSQL</li>
  <li>Bootstrap</li>
</ul>

### Setup

1. Clone the repo
  ```bash
  git clone https://github.com/serginavarra1/Expense-Tracker.git
  ```
2. Install NPM Packages
  ```bash
  npm install
  ```
3. Start the server
 ```bash
 npm run dev
 ```
